

# AI-Stylist Landing Page

A minimal and elegant landing page for your AI-powered stylist mobile app, featuring taupe/earth tones on a light background.

---

## Design Direction
- **Theme**: Light, clean, minimal and elegant
- **Colors**: Taupe and earth tones (#8b7e74, warm beiges, soft creams)
- **Typography**: Modern sans-serif, clean hierarchy
- **Overall feel**: Sophisticated, fashion-forward, trustworthy

---

## Page Structure

### 1. Header / Navigation
- AI-Stylist logo (text-based)
- Simple navigation: Features, How It Works, Join Waitlist button
- Clean, minimal sticky header

### 2. Hero Section
- **Headline**: Compelling tagline about AI-powered personal styling
- **Subheadline**: Brief description of what the app does
- **Phone mockup**: Elegant display of the app interface
- **CTA Button**: "Join the Waitlist"
- **Email input field** for waitlist signup

### 3. Features Section
Showcase the 5 core features with icons:
1. **Kibbe Body Type Analysis** - Personalized style recommendations based on your body type
2. **Digital Wardrobe** - Save and organize all your clothes in one place
3. **Smart Outfit Combinations** - AI-suggested outfits from your existing wardrobe
4. **Virtual Try-On** - See how clothes look on you before buying
5. **Shop Links** - Find and purchase recommended items instantly

### 4. CTA / Waitlist Section
- Repeat the value proposition
- Email signup form
- "Join the Waitlist" button
- Optional: Show waitlist count or "Join X+ others"

### 5. Footer
- Logo
- Social links (placeholder)
- Copyright

---

## Interactions
- Smooth scroll navigation
- Subtle hover animations on buttons and feature cards
- Email validation for waitlist form
- Success toast notification when joining waitlist
